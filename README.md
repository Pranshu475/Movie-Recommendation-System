# Movie-Recommendation-System

## Overview
This project is an **Item-Based Collaborative Filtering** system built with Python. It analyzes user ratings to suggest movies similar to a user's favorites.

## How It Works
* **Data:** Uses the MovieLens Small Dataset (100k ratings).
* **Logic:** Converts user ratings into a Matrix and calculates **Pearson Correlation** to find similarity scores.
* **Visualization:** Generates Heatmaps using Seaborn to visualize movie relationships.

## Tech Stack
* Python (Pandas, NumPy)
* Visualization (Seaborn, Matplotlib)
